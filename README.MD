# play

Функция для игры "Виселица". На вход получает заданное слово. 
У пользователя 6 попыток чтоб угадать слово. Если заданное слово пустое, оно будет иметь пустое значение
Код будет работать пока пользователь не исчерпает все 6 попыток. Имеется проверка на пустой ввод 
Основной код этой функции отвечает за саму игру. Если пользователь угадал букву то она попадает в словарь с угаданными буквами, если нет, то в словарь с неудачными попытками.
Так же после неудачи пользователь теряет одну попытку. После потери всех 6 попыток функция вывод в консоль информацию о проигрыше и правильный ответ
